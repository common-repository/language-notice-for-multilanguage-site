== Installation ==

The plugin is simple to install:

1. Download 'language-notice-for-multilanguage-site.zip'
2. Unzip
3. Upload 'language-notice-for-multilanguage-site' directory to your '/wp-content/plugins' directory, or install the plugin throught the Wordpress plugins screen directly.
4. Activate the plugin through the 'Plugins' menu in WordPress.
